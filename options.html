QuickTrader

What is QuickTrader?

QuickTrader is a Google Chrome extension made to assist day traders using TD Direct Investing, find and fill the
best limit price for stocks and option contracts for faster trade execution through the TD WebBroker Dashboard.

How to Use the Extension:

In order for QuickTrader to automatically find and fill the price of the security the user is trading, they must:

For Stocks:

  • Populate the ticker symbol field
  • Change the price type to 'Limit'
For Option Contracts:
  • Populate the ticker symbol field
  • Populate the contract type field
  • Populate the contract expiry date field
  • Populate the strike price field
  • Change the price type to 'Limit'
  • Tip: Use the Option Chain (Fills in all the contract info at once)

If all of the required Fields are filled the extension will find and fill the price of preference.
It can also update the price every time you press the referesh button on the order page.

Changing The Fill Type:
If the user chooses to change the fill type, it can be done through the extension's popup.

  1. Click on the extension icon to open the popup window.
  2. Select the fill type of choice
  3. Click save
This will save the fill type until changed or until chrome is closed.
** Default fill type is the middle price (between the bid and ask) if nothing is selected

Tips for ENTERING trades:

  • For the best return on investment, set fill type to 'Bid' or 'Above Bid' (Lower chance of getting filled - Lower risk)
  • To play it safe, set fill type to 'Middle Price' (Good chance of getting filled - Balanced risk)
  • To enter a position fast for fast moving stocks, set fill type to 'Below Ask' or 'Ask' (Higher chance of getting filled - Higher risk)

    • Tips for EXITING trades:

  • For the best return on investment, set fill type to 'Below Ask' or 'Ask' (Lower chance of getting filled - Higher profits)
  • To play it safe, set fill type to 'Middle Price' (Good chance of getting filled - Balanced profits)
  • To exit a position fast, set fill type to 'Above Bid' or 'Bid' (Higher chance of getting filled - Lower profits)

    • Disclaimers:
    The tips are not to be used without prior experience/understanding of the security price movement.
    Results will vary depending on entry/exit prices aswell as trader's risk tolerance.
